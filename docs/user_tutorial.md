## 1. 시작하기

Fashion AI Generator에 오신 것을 환영합니다! AI 기반 패션 트렌드 분석 및 디자인 생성 플랫폼을 통해 창의적인 패션 디자인을 손쉽게 만들어보세요.

### 1.1. 시스템 요구사항
- **웹 브라우저**: Chrome, Firefox, Safari, Edge 최신 버전
- **인터넷**: 안정적인 고속 인터넷 연결 (10Mbps 이상 권장)
- **해상도**: 최소 1024x768, 권장 1920x1080 이상

### 1.2. 회원가입
1. [fashion-ai.com](https://fashion-ai.com) 접속
2.右上角 "회원가입" 클릭
3. 필요 정보 입력:
   - 사용자 이름
   - 이메일 주소
   - 비밀번호 (8자 이상, 영문+숫자+특수문자)
4. 이메일 인증 완료
5. 로그인 후 바로 시작!

## 3. 트렌드 분석 사용법

### 3.1. 기본 트렌드 분석

#### 단계 1: 키워드 입력
1. "트렌드 분석" 탭 클릭
2. 키워드 입력창에 관심 키워드 입력
   - 예시: "오버사이즈", "지속가능패션", "2025 S/S"
   - 최대 10개까지 동시 분석 가능

#### 단계 2: 분석 옵션 설정
```
□ 분석 기간: ▼ 최근 7일 ▼
□ 데이터 소스:
☑ 패션 뉴스
☑ 인스타그램
☑ 무신사
☑ WGSN
□ 상세 옵션:
☑ 감성 분석 포함
☑ 시각적 트렌드 분석
☑ 관련 키워드 추출
```

#### 단계 3: 분석 시작 및 결과 확인
1. "분석 시작" 버튼 클릭
2. 진행률 표시 (보통 3-10분 소요)
3. 분석 결과 대시보드 표시

### 3.2. 분석 결과 해석

#### 트렌드 스코어
- **0-25점**: 하락 트렌드
- **26-50점**: 유지 트렌드
- **51-75점**: 상승 트렌드
- **76-100점**: 폭발적 트렌드

#### 감성 분석
- **긍정적 (초록)**: 호평이 많은 트렌드
- **중립 (노랑)**: 평범한 반응
- **부정적 (빨강)**: 비판적인 의견 많음

#### 관련 키워드 구름
- 단어 크기 = 관련성 높음
- 색상 = 성장률 (빨강: 높음, 파랑: 낮음)

### 3.3. 고급 분석 팁

1. **시계열 분석**
   - "1개월" 선택으로 장기 추이 파악
   - 계절성 패턴 발견

2. **비교 분석**
   - 복수 키워드 동시 입력
   - 트렌드 간 상관관계 파악

3. **시각 트렌드**
   - 색상 트렌드: 주요 컬러 파레트 제안
   - 실루엣 트렌드: 인기 있는 실루엣 형태

## 4. 이미지 생성 사용법

### 4.1. 텍스트 기반 디자인 생성

#### 단계 1: 디자인 컨셉 설명
```
디자인 설명: __________________________________________
예시: "미니멀한 A라인 미디 원피스, 순면 소재,
     베이지색, 라운드 넥라인, 앞면 디테일 없음"
```

#### 단계 2: 상세 옵션 선택
```
스타일:          ▼ 모던 ▼
의류 타입:       ▼ 원피스 ▼
주요 색상:       [베이지] [+] [화이트] [+]
소재:            ▼ 면 ▼
생성 수량:       (3)개
해상도:          ○ 표준 1024x1024
                ● 고품질 2048x2048
                ○ 초고품질 4096x4096
```

#### 단계 3: 고급 설정
```
□ 참조 이미지 사용: [파일 선택]
□ 특정 디자이너 스타일: ▼
□ 타겟 고객층:       20대 여성
□ 제작 용도:        데일리웨어
```

#### 단계 4: 생성 및 결과
1. "생성 시작" 버튼 클릭
2. 약 1-3분 후 결과 표시
3. 각 이미지에 대해:
   - [❤️] 좋아요
   - [💾] 저장
   - [🔄] 유사 디자인
   - [✏️] 수정 요청

### 4.2. 이미지 기반 생성

#### 이미지 업로드 방식
1. 참조할 패션 이미지 업로드
2. 수정할 부분 선택:
   - 전체 유지, 색상만 변경
   - 실루엣 유지, 디테일 변경
   - 부분 수정 (소매, 칼라 등)

#### 수동 수정 요청
```
수정할 부분: __________________________________
예시: "소매를 7부소매로 변경해주세요.
     넥라인을 V넥으로 바꿔주세요."
```

### 4.3. 컬렉션 생성

#### 컬렉션 설정
```
테마: "2025년 여름 리조트웨어"
포함 의류: ☑ 상의 ☑ 하의 ☑ 원피스 ☑ 수영복
통합 색상: [아쿠아] [코랄] [샌드]
스타일: 레지던스
세트 수량: 5개
```

#### 결과물
- 각 세트별 조합 제안
- 색상 배리에이션
- 소재 믹스매치 가이드

## 5. 패턴 생성 사용법

### 5.1. 디자인을 패턴으로 변환

#### 패턴 생성 요청
1. 생성된 디자인 선택
2. 패턴 옵션 설정:
```
의류 타입:      ▼ 블라우스 ▼
사이즈 시스템:   ▼ KS (한국) ▼
사이즈:         ▼ M ▼
필수 치수:
- 가슴 둘레:  84cm
- 허리 둘레:  68cm
- 엉덩이 둘레: 92cm
□ 맞춤 제작 여유 추가
```

### 5.2. 패턴 결과물

#### 패턴 조각
- **앞판**: 앞 몸판 패턴
- **뒤판**: 뒤 몸판 패턴
- **소매**: 소매 패턴 (있는 경우)
- **기타**: 칼라, 커프스 등

각 조각에는:
- 실제 크기 도면 (1:1)
- 재단선 및 시접선
- 치수 표시
- 제작 가이드

#### 제작 설명서
```
1. 재단
   - 시접 1.5cm 포함하여 재단
   - 직물 방향 화살표 따르기

2. 표시
   - 다트 위치 표시
   - 단추 구멍 위치 마킹

3.缝合 순서
   1) 어깨솔기
   2) 옆솔기
   3) 소매 달기
   ...
```

### 5.3. 사이즈 그레이딩

#### 다중 사이즈 생성
```
기준 사이즈: M
추가 사이즈: ☑ S ☑ L ☑ XL
그레이딩 규칙: ▼ 표준 규칙 ▼
□ 맞춤 규칙 사용
```

#### 결과
- 각 사이즈별 패턴 자동 생성
- 사이즈별 치수표
- 재단 계획도

## 9. 문제 해결

### 9.1. 일반적인 문제

#### Q: 이미지 생성이 너무 오래 걸려요
A:
- 해상도를 표준으로 낮춰보세요
- 동시 요청 수를 줄여보세요
- 새로고침 후 다시 시도

#### Q: 생성된 디자인이 마음에 안 들어요
A:
- 프롬프트를 더 구체적으로 수정
- 참조 이미지 활용
- 스타일 옵션 변경

#### Q: 패턴이 부정확해요
A:
- 정확한 치수 입력 확인
- 사이즈 시스템 확인
- 제작 가이드 재확인

### 9.2. 기술적인 문제

#### 로딩 오류
1. 브라우저 캐시 삭제
2. 다른 브라우저로 시도
3. 인터넷 연결 확인

#### 이미지 표시 문제
1. 팝업 차단 해제
2. 이미지 자동 로드 설정 확인
3. VPN 끄기
