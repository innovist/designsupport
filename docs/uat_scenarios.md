## 2. 기능 테스트 시나리오

### 2.1. 트렌드 분석 기능

#### TC001: 기본 트렌드 분석
**테스트 목적**: 키워드 기반 트렌드 분석 기능 검증

**수행 단계**:
1. 메인 페이지 접속
2. 언어를 한국어로 설정
3. "트렌드 분석" 탭 클릭
4. 키워드 입력: "오버사이즈 블레이저"
5. 기간 선택: "최근 7일"
6. "분석 시작" 버튼 클릭
7. 결과 확인

**예상 결과**:
- 로딩 인디케이터 표시
- 10분 내에 분석 결과 표시
- 트렌드 점수, 관련 키워드, 시각화 차트 포함
- 결과 데이터 다운로드 가능

#### TC002: 다중 키워드 분석
**테스트 목적**: 여러 키워드 동시 분석 기능 검증

**수행 단계**:
1. 트렌드 분석 페이지 접속
2. 키워드 입력: "미니멀리즘", "지속가능 패션"
3. 기간 선택: "최근 30일"
4. 분석 시작

**예상 결과**:
- 각 키워드별 분석 결과 제공
- 키워드 간 상관관계 시각화
- 트렌드 비교 분석 제공

### 2.2. 이미지 생성 기능

#### TC003: 기본 디자인 생성
**테스트 목적**: 텍스트 프롬프트 기반 이미지 생성 검증

**수행 단계**:
1. "이미지 생성" 탭 클릭
2. 디자인 설명 입력: "미니멀한 A라인 원피스, 흰색, 린넨 소재"
3. 스타일 선택: "모던"
4. 의류 타입: "원피스"
5. 색상: "화이트"
6. 소재: "린넨"
7. 생성 버튼 클릭

**예상 결과**:
- 5개 이상의 디자인 변형 생성
- 생성 시간 2분 내외
- 고해상도 이미지 (1024x1024)
- 다운로드 및 저장 기능

#### TC004: 참조 이미지 기반 생성
**테스트 목적**: 이미지 레퍼런스 기반 디자인 생성 검증

**수행 단계**:
1. 참조 이미지 업로드 (기존 패션 이미지)
2. "유사한 디자인 생성" 옵션 선택
3. 수정 요청 입력: "소매를 7분의 3소매로 변경"
4. 생성 실행

**예상 결과**:
- 참조 이미지의 특징 유지
- 요청된 수정사항 반영
- 일관성 있는 디자인 결과

### 2.3. 패턴 생성 기능

#### TC005: 도면 생성
**테스트 목적**: 디자인을 실제 패턴 도면으로 변환 검증

**수행 단계**:
1. "패턴 생성" 탭 클릭
2. 생성된 디자인 이미지 선택
3. 의류 타입: "블라우스"
4. 사이즈 시스템: "KS (한국)"
5. 사이즈: "M"
6. "패턴 생성" 클릭

**예상 결과**:
- 앞/뒤/소매 패턴 조각 생성
- 상세 치수 표시
- 봉제선 및 시접 여유 포함
- PDF 다운로드 가능

#### TC006: 사이즈 그레이딩
**테스트 목적**: 다양한 사이즈의 패턴 생성 검증

**수행 단계**:
1. 기존 패턴 선택
2. 사이즈 범위: S, M, L, XL
3. "사이즈 그레이딩" 실행

**예상 결과**:
- 각 사이즈별 패턴 정확한 생성
- 비례 유지
- 그레이딩 룰 적용
