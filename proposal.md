# AI 기반 패션 트렌드 분석 및 디자인 자동 생성 시스템 구축 제안서

---

## 0. 표지

**AI 기반 패션 트렌드 분석 및 디자인 자동 생성 시스템 구축 제안**

Data-Driven Fashion Design & Trend Forecasting System

From Data to Blueprint in Minutes

2025.12

---

## 1. The Question (문제 인식)

"매주 수많은 신상품이 쏟아지는 시대, 아직도 디자이너의 감에만 의존하고 계십니까?"

- 트렌드 주기는 짧아졌고, 정보는 폭증했습니다.
- 수집 업무는 늘어났으나, 기획과 디자인에 쓰이는 시간은 줄었습니다.
- 결과적으로 시장 타이밍을 놓치거나 재고 리스크가 커지는 문제가 발생합니다.

---

## 2. Executive Summary (요약)

본 제안은 **패션 트렌드 데이터를 자동 수집·분석**하고, 이를 기반으로 **디자인 아이디어, 이미지, 블루프린트(도면)**까지 한 번에 생성하는 통합 AI 시스템 구축을 제안합니다.

핵심 가치는 다음과 같습니다.
- 데이터 수집부터 결과 생성까지 **끝단까지 연결된 파이프라인**
- 트렌드 분석 결과가 즉시 디자인 산출물로 이어지는 **기획-제작 연계**
- 프로젝트/세션 기반 자산 축적과 재활용

---

## 3. 시장 맥락 및 Pain Point

### 3.1 트렌드 주기의 붕괴
- 시즌 경계가 흐려지고 마이크로 트렌드가 단기간에 확산됩니다.
- SNS/커뮤니티 기반 신호를 빠르게 읽지 못하면 경쟁에서 뒤처집니다.

### 3.2 정보 과잉과 기획 병목
- 디자이너가 여러 채널을 직접 조사해야 하는 구조입니다.
- 수집은 많지만, **의미 있는 인사이트로 연결되는 과정**이 부족합니다.

---

## 4. 제안 솔루션 개요

### 4.1 AI Co-Designer 컨셉
본 시스템은 단순 이미지 생성 툴이 아니라, **시장 조사부터 초기 생산 문서까지 연결하는 지능형 에이전트**입니다.

- Trend Watcher: 데이터 수집과 신호 탐지
- Data Analyst: 트렌드/댓글 인사이트 분석
- Pattern Maker: 도면/패턴 산출

---

## 5. 핵심 경쟁력 (차별점)

| 구분 | 일반 AI 이미지 생성 | 제안 시스템 |
| --- | --- | --- |
| 데이터 소스 | 없음 (프롬프트 의존) | 크롤러 + 검색 기반 수집 |
| 분석 능력 | 없음 | LLM 기반 트렌드/인사이트 분석 |
| 결과물 | 단순 이미지 | 이미지 + 도면 + 리포트 |
| 활용성 | 아이디어 참고용 | 프로젝트/샘플 협업까지 확장 |

---

## 6. 시스템 아키텍처

### 6.1 구성 요소
- **Frontend**: Jinja2 템플릿, Vanilla JS, CSS
- **Backend**: FastAPI (Python), 비동기 파이프라인
- **AI Core**: Gemini, GLM (텍스트), Seedream/Nano Banana/Z-Image (이미지)
- **Storage**: SQLite (기본), 파일 저장소, 설정 파일
- **검색 확장**: SearXNG 로컬 연동

### 6.2 주요 모듈
- Pipeline Orchestrator: 전체 파이프라인 제어
- Analysis Service: 다중 모델 분석 및 종합
- Report Generation: 리포트 JSON 생성 및 저장
- Image/Blueprint Service: 이미지 및 도면 생성
- Crawler Service: 크롤러 오케스트레이션

## 7. 데이터 수집 설계

### 7.1 크롤러 카탈로그 (카테고리 기준 21종)
- 패션 플랫폼: 무신사, 인스타그램, 핀터레스트
- 커뮤니티: DC인사이드, 더쿠, FM코리아, 클리앙, 루리웹, 뽐뿌, MLB파크, 블라인드, 에토랜드, 인벤
- 포털/블로그: 네이버 블로그, 네이버 카페, 다음 카페
- 미디어: 유튜브
- 뉴스/언론: 패션N, WGSN, 네이트뉴스
- 검색: SearXNG

### 7.2 기본 파이프라인 등록 크롤러
- 패션N, 인스타그램, 무신사, WGSN, 핀터레스트, 유튜브, 네이트뉴스
- 추가로 SearXNG 자동 등록

> 필요 시 신규 크롤러 등록만으로 확장 가능합니다.

## 8. 파이프라인 (8단계)

1. 입력 분석
- 세션 설명/URL/파일을 통합해 컨텍스트 구성

2. 키워드 추출
- Gemini 기반 키워드 생성, 실패 시 GLM 폴백

3. 크롤링 수집
- 키워드별 병렬 수집, 기간/소스/유튜브 채널 설정 반영

3.5. AI 직접 조사 (선택)
- Gemini Search, Perplexity, GLM 웹 리서치 병렬 수행

4. 트렌드 분석
- 수집 데이터 + 댓글 인사이트 분석
- 다중 모델 결과 통합

5. 디자인 아이디어 생성
- 분석 결과 기반 3~5개 컨셉 도출

6. 리포트 생성
- JSON 기반 리포트 생성 및 저장
- 다국어 지원 (ko/en/zh-CN/zh-TW)

7. 이미지 생성
- 마스터 디자인(Flat Lay) + 모델 착장 이미지

8. 블루프린트 생성
- 스케치/레이아웃/패턴 3종 생성
